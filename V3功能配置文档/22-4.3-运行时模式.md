# 4.3 运行时模式（说明）

## 模式列表
- useSingleGridRuntime：单表页面
- useMasterDetailRuntime：主从页面
- useBaseRuntime：基础运行时（可自定义 features）

## 配置关联
- **单表页面**：只需一个 GRID 组件；不需要 TABS/MasterDetail
- **主从页面**：需要 GRID + TABS +（可选）MASTER_DETAIL 容器

## 示例：单表页面组件树
```sql
INSERT INTO T_COST_PAGE_COMPONENT (ID, PAGE_CODE, COMPONENT_KEY, COMPONENT_TYPE, REF_TABLE_CODE, SORT_ORDER, CREATE_BY)
VALUES (SEQ_COST_PAGE_COMPONENT.NEXTVAL, 'cost-single', 'masterGrid', 'GRID', 'CostMaster', 0, 'system');
```

## 注意事项
- 运行时模式为前端代码选择；数据库配置只决定组件树与规则内容。
- 若页面包含 TABS / MASTER_DETAIL，建议使用 useMasterDetailRuntime。
