# 1.6 右键菜单（Context Menu）配置说明

## 配置入口
- **T_COST_PAGE_RULE**，ruleType = `CONTEXT_MENU`
- componentKey：`masterGrid`（主表）或具体 tabKey（明细）

## 规则结构
```json
{
  "items": [
    {"action":"addRow","label":"新增"},
    {"action":"copyRow","label":"复制","requiresRow":true},
    {"type":"separator"},
    {"action":"deleteRow","label":"删除","requiresRow":true},
    {"action":"save","label":"保存"}
  ]
}
```

## 支持的内置 action
- addRow / copyRow / deleteRow / save
- saveGridConfig（保存列配置）
- exportSelected / exportCurrent / exportAll / resetExportConfig / openHeaderConfig
- clipboardCopy / clipboardPaste
- separator

> 未识别的 action 会被忽略（对部分可选 action 不会报警）。

## 自定义 Action（存储过程/SQL/Java）

除了内置 action，还可以在菜单项中直接配置存储过程、SQL 或 Java 方法调用：

### 存储过程调用
```json
{
  "action": "generateFromBom",
  "label": "由BOM生成明细",
  "requiresRow": true,
  "procedure": "P_COST_BOM_INSERT",
  "params": [
    {"source": "data.docid", "mode": "IN", "jdbcType": "NUMERIC"}
  ]
}
```

### SQL 执行
```json
{
  "action": "approve",
  "label": "审核",
  "requiresRow": true,
  "sql": "UPDATE T_COST_MASTER SET STATUS = '已审核' WHERE ID = ${id}"
}
```

### Java 方法调用
```json
{
  "action": "syncData",
  "label": "同步数据",
  "handler": "costActionHandler.sync"
}
```

### params 参数说明
| 字段 | 说明 | 示例 |
|------|------|------|
| source | 参数来源，`data.字段名` 从当前行取值 | `data.docid` |
| target | OUT 参数输出目标 | `vars.result` |
| mode | 参数模式：IN / OUT / INOUT | `IN` |
| jdbcType | JDBC 类型 | `NUMERIC`, `VARCHAR` |

## 示例 A：主表右键菜单（含自定义 action）
```sql
INSERT INTO T_COST_PAGE_RULE (ID, PAGE_CODE, COMPONENT_KEY, RULE_TYPE, RULES, CREATE_BY)
VALUES (
  SEQ_COST_PAGE_RULE.NEXTVAL,
  'cost-pinggu',
  'masterGrid',
  'CONTEXT_MENU',
  '{"items":[
    {"action":"addRow"},
    {"action":"copyRow","requiresRow":true},
    {"action":"deleteRow","requiresRow":true},
    {"type":"separator"},
    {"action":"generateFromBom","label":"由BOM生成明细","requiresRow":true,"procedure":"P_COST_BOM_INSERT","params":[{"source":"data.docid","mode":"IN","jdbcType":"NUMERIC"}]},
    {"type":"separator"},
    {"action":"saveGridConfig"},
    {"type":"separator"},
    {"action":"save"}
  ]}',
  'system'
);
```

## 示例 B：明细 Tab 右键菜单
```sql
INSERT INTO T_COST_PAGE_RULE (ID, PAGE_CODE, COMPONENT_KEY, RULE_TYPE, RULES, CREATE_BY)
VALUES (
  SEQ_COST_PAGE_RULE.NEXTVAL,

    "items": [

      {"action":"clipboardPaste","label":"粘贴单元格"}
```

## 多种配置情况
1) **分组菜单**
```json
{
  "items": [
    {"label":"编辑","type":"group","items":[{"action":"addRow"},{"action":"deleteRow"}]},
    {"label":"导出","type":"group","items":[{"action":"exportCurrent"},{"action":"exportAll"}]}
  ]
}
```
2) **动态禁用**：`disabled:true` 或 `requiresSelection:true`

## 注意事项
- 明细菜单按 tabKey 生效；未配置的 tab 使用空菜单。
- 自定义导出菜单会自动追加（当存在自定义导出配置时）。
- 自定义 action 执行成功后会自动刷新表格数据。
