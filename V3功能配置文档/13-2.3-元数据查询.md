# 2.3 元数据查询（使用说明）

## 接口列表
- GET `/api/metadata/table/{tableCode}`：查询表元数据（不含权限）
- GET `/api/metadata/table/{tableCode}/page/{pageCode}?gridKey=masterGrid`：查询表元数据（合并权限与列偏好）
- GET `/api/metadata/page/{pageCode}`：页面组件树 + 页面规则
- GET `/api/metadata/dict/{dictType}`：字典项
- GET `/api/metadata/lookup/{lookupCode}`：Lookup 配置
- POST `/api/metadata/cache/clear?tableCode=CostMaster`：清除缓存

## 示例 A：获取表元数据（含权限）
```
GET /api/metadata/table/CostMaster/page/cost-demo?gridKey=masterGrid
```

## 示例 B：获取页面组件树
```
GET /api/metadata/page/cost-demo
```

## 注意事项
- 权限版接口要求用户登录，并且 pageCode 有权限。
- 列偏好会在权限版接口中自动合并（按 userId + pageCode + gridKey）。
