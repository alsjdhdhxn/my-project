# 4.2 响应式架构（说明）

## 说明
该项为前端运行时架构（Vue3 Composition API + Ref/Reactive），**不依赖数据库配置**。

## 可做的“可配置点”（非 DB）
- 通过运行时 features 开关启用/禁用某些能力（detailTabs / broadcast / aggregates / lookup / export / contextMenu）。
- 通过 componentStateByKey 控制组件状态（ready/loading/error）。

## 示例（代码层）
```ts
const runtime = useBaseRuntime({ pageCode, notifyInfo, notifyError, notifySuccess }, {
  detailTabs: true,
  broadcast: true,
  aggregates: true,
  lookup: true,
  export: true,
  contextMenu: true
});
```

## 注意事项
- 如需完全 DB 控制，请通过页面规则 / 组件配置实现，不建议在此层硬编码。
