## 发版
powershell -ExecutionPolicy Bypass -File .\scripts\deploy.ps1
## 查看传输
powershell -ExecutionPolicy Bypass -File .\scripts\deploy.ps1 -DryRun
## 只同步文件不重建容器
powershell -ExecutionPolicy Bypass -File .\scripts\deploy.ps1 -SkipRebuild

## 备注
- 后端 Dockerfile 使用 BuildKit cache 挂载缓存 Maven 依赖：`/root/.m2`。
- 前端 Dockerfile 使用 BuildKit cache 挂载缓存 pnpm 依赖：`/pnpm/store`。
- 这些缓存只用于依赖下载，不缓存后端源码、不缓存前端页面源码、不缓存前端 `dist` 构建产物。
- 如果怀疑 Maven 或 pnpm 依赖缓存异常，可在服务器执行 `docker builder prune` 清理构建缓存后重新发版。
- 如果怀疑前端页面一直不变化，优先排查浏览器/Nginx/前端镜像构建缓存；依赖缓存本身不会复用旧页面代码。
